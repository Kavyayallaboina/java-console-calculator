This Java program allows users to perform *addition, subtraction, multiplication, and division* operations.  
It uses a Scanner for input and includes a looping mechanism that lets users perform multiple calculations until they choose to exit.

# Features
- Written in *pure Java*
- Uses *methods* for each arithmetic operation  
- Includes *error handling* for division by zero  
- Allows multiple calculations in one run  
- Clean and beginner-friendly structure
